# 剧情目录索引

> 当前契约（Stage 3+ 权威产物齐备）。本目录基于《盘龙》开篇第 1-23 章节选，不代表全书结论。

| 文件 | 权威范围 | 下游用法 |
|---|---|---|
| `节奏.md` | 关键信息推进、爽点循环、情绪触动点、爆发节奏 | 产出 `rhythm_reference` |
| `情绪模块.md` | 读者需求、情绪引擎、套路框架、可复现模块卡 | 产出 `selected_emotion_module` |
| `故事线.md` | 故事框架与故事线摘要 | 投影摘要，不覆盖权威文件 |
| `拆文报告.md` | 人类阅读报告 | 投影摘要，不覆盖权威文件 |

冲突时以 `节奏.md` / `情绪模块.md` 为准；两个权威文件用 EM/RH/TR ID 互相引用。
