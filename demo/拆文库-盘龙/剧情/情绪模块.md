# 情绪模块：盘龙（开篇第 1-23 章节选）

> 权威范围：读者需求 / 情绪引擎 / 套路框架运转 / 可复现模块。禁止把原文具体桥段当模板照搬。

## 读者需求 / 情绪引擎

| 读者需求 | 本书满足方式 | 证据章节/情节点 | 持续追读机制 | 可迁移边界 |
|---|---|---|---|---|
| 成长陪伴与努力被看见 | 用晨练、苦修、身体反应和导师认可连续证明主角投入，而不是直接宣告天才 | 第 1 章、TR-001；第 20-23 章、TR-006 | 每次认可后立更高能力门槛 | 可保留“行动 → 认可 → 新目标”；必须替换训练体系与人物 |
| 身份补偿与家族复兴 | 先给衰败处境，再揭示辉煌血脉；检测失败使荣耀成为待完成目标 | 第 2-3 章、RH-001 | 赎回家族象征、验证血脉潜力等长线问题持续悬置 | 可保留“辉煌过去 vs 困顿现在”；不得照搬血脉、家族名与具体遗物 |
| 对力量上限的敬畏与向往 | 由导师示范、强者过境、圣域之战分层展示天花板 | 第 1、8-15 章，TR-003 | 每次看见更高层级后，主角目标随之升级 | 强者展示必须服务主角选择；不可复制能力名称和事件顺序 |
| 亲情守护与善有回报 | 父子共同承受家族落差，兄长在灾难中护弟并付出受伤代价 | 第 3、16-18 章，TR-002、TR-004 | 读者等待情感选择如何转化为成长与关系回报 | 可保留“选择 → 代价 → 回报”因果；不可机械套用受伤认主 |
| 认知反转与延迟满足 | 先把盘龙戒指写成廉价日常物，跨章后再翻成命运机缘 | 第 6-7、17-18 章，RH-002、TR-005 | 读者等待已埋物件何时异变、异变后带来什么规则 | 可保留功能跃迁；必须替换专名、外形、触发方式和奖励内容 |

## 故事框架与套路运行图

| 框架/套路 | 在本书中的位置 | 运行方式 | 为什么有效 | 失效风险 |
|---|---|---|---|---|
| 慢热升级流 | 第 1-23 章主线 | 立力量天花板 → 暴露主角缺口 → 多轮苦练与认知升级 → 机缘改变路线 → 初次能力兑现 | 成长尺度清楚，情感投资能在机缘释放时复利 | 首次强释放过晚，面向快节奏平台容易流失 |
| 衰败家族复兴 | 第 2-3 章及后续长线目标 | 揭示辉煌过去 → 当下检测失败 → 主角主动继承目标 | 身份荣耀和现实困顿形成稳定张力 | 只有设定没有可见行动时会变成口号 |
| 掉价物件延迟翻面 | 第 6-18 章 | 日常低估 → 长期搁置 → 灾难中异变 → 身份与功能集中揭示 | 早期轻描淡写会放大后期认知反转 | 提示过重会泄底；延迟过长而无中间回扣会遗忘 |
| 善行触发机缘 | 第 16-19 章 | 危机加压 → 主角主动护人并受伤 → 既有伏笔被触发 → 新成长路线开启 | 奖励与人物品质形成因果，不像无条件送挂 | 若代价虚假或回报过快，会显得功利和机械 |
| 导师授业升级 | 第 18-23 章 | 导师身份确认 → 解释规则 → 检测天赋 → 立训练要求 → 首次兑现 | 设定、关系和任务同时推进 | 长段讲解缺行动验证时会拖慢节奏 |

## 可复现模块卡

### EM-001 天花板先行的成长纵深

| 字段 | 内容 |
|---|---|
| 读者想看什么 | 弱小主角看见可抵达但遥远的高度，并用行动开始追赶。 |
| 情绪链 | 当下弱小 → 目睹上限 → 产生敬畏与渴望 → 主动作出选择 → 新门槛出现 |
| 戏剧单元 | 让可信的强者或事件展示上限，再让主角以具体行动而非口号回应。 |
| 关键触发物 | 上限展示者、可量化差距、在场见证者、主角的第一步行动 |
| 复现步骤 | 1. 建立当前能力基线；2. 展示高一至两层的上限；3. 写出主角身体或行为反应；4. 落到可执行的新目标。 |
| 可替换项 | 力量体系、职业、展示场景、导师身份、第一步任务 |
| 不可照搬 | 四大终极战士、圣域等级、乌山镇晨练及相关专名 |
| 关联节奏 | `剧情/节奏.md` RH-001 / TR-001 / TR-003 |

### EM-002 低估物件的跨章功能跃迁

| 字段 | 内容 |
|---|---|
| 读者想看什么 | 早期被所有人低估的东西，在关键时刻显出完全不同的价值。 |
| 情绪链 | 日常获得 → 主角与读者共同低估 → 中段微异变 → 真相爆发 → 新问题立起 |
| 戏剧单元 | 同一物件先承担人设或日常功能，再承担暗线提示，最后承担命运转折。 |
| 关键触发物 | 被低估物、错误判断、无人察觉的异变、身份确认者、新功能代价 |
| 复现步骤 | 1. 先让物件在当下场景有普通用途；2. 给出可信的低估理由；3. 跨章回扣一次异变；4. 在角色选择或危机后揭示真正功能。 |
| 可替换项 | 物件形态、普通用途、触发条件、揭示者、真正能力 |
| 不可照搬 | 盘龙戒指、龙纹外形、滴血认主、戒中圣域魔导师及原事件顺序 |
| 关联节奏 | `剧情/节奏.md` RH-002 / TR-005 |

### EM-003 选择付代价后触发机缘

| 字段 | 内容 |
|---|---|
| 读者想看什么 | 主角先因品格作出高成本选择，再获得与该选择有因果关系的回报。 |
| 情绪链 | 危机逼近 → 主角主动承担 → 真实代价落下 → 旧伏笔被触发 → 希望与新责任并存 |
| 戏剧单元 | 机缘不是随机掉落，而是由主角保护他人、守诺或承担责任的行为开启。 |
| 关键触发物 | 被保护者、不可回避的代价、已埋伏笔、触发条件、回报后的新责任 |
| 复现步骤 | 1. 先埋可触发资源；2. 把角色放进必须选择的危机；3. 让选择造成可见代价；4. 再触发资源；5. 回报后立即建立责任或训练要求。 |
| 可替换项 | 危机类型、保护对象、代价、伏笔载体、回报方向 |
| 不可照搬 | 护弟受伤、鲜血染戒、导师出戒等具体动作链 |
| 关联节奏 | `剧情/节奏.md` RH-003 / TR-004 / TR-005 |

### EM-004 导师讲解立刻接行动验证

| 字段 | 内容 |
|---|---|
| 读者想看什么 | 快速理解新体系，并马上看到主角在体系中的位置和进展。 |
| 情绪链 | 规则未知 → 类比理解 → 天赋悬念 → 结果确认 → 苦练与首次兑现 |
| 戏剧单元 | 导师只讲当前任务需要的规则，随后用测试、训练或失败立即验证。 |
| 关键触发物 | 导师、生活化类比、测试标准、训练任务、首次可见成果 |
| 复现步骤 | 1. 由主角问题触发讲解；2. 用熟悉事物类比；3. 给测试标准；4. 当场验证；5. 用一次不完美成果收束。 |
| 可替换项 | 知识体系、类比对象、测试方式、训练周期、初次成果 |
| 不可照搬 | 地系魔法、元素亲和力数值、精神力倍数、撼地术与师徒专名 |
| 关联节奏 | `剧情/节奏.md` RH-004 / TR-006 |

## 重组与复现指南

1. 先选读者需求，再选模块卡，不要从原文桥段倒抄。
2. 保留情绪链和功能位，替换人物、场景、动机、道具、事件素材。
3. 复现到新书大纲时，至少改变冲突对象、公开/私密场景、触发证据、余波方向中的三项。
4. 若人设与模块冲突，以人设驱动重构模块；不要为了套套路扭曲角色。
