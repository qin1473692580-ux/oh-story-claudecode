# 细纲（第 18 章）

### 第18章：全民热议
- 阶段位置：高潮期 · 第2阶段第8章；兑现反派代价并转入全民善意余波。
- 本章结构公式：吴伟遭反向围攻 + 删除道歉无效 + 账号永久封禁 + 公诉通知与全民慰问
- 本章禁止提前释放：作品在军营、校园和老兵群体中的后续回响；本章只完成吴伟代价兑现。
- 核心事件：
  - 《如愿》点赞转发量呈几何级暴增，即将破百万，官方媒体转发认可
  - 吴伟打开自己账号，发现网友已涌入主页大量恶评轰炸
  - 吴伟拼命删除之前诋毁江晨的视频，但截图和二次转发铺天盖地，无法根除
  - 平台发出警告提示：大量用户举报吴伟'恶意诽谤'
  - 吴伟被迫录制道歉视频，语气表情生硬，仍遭评论区嘲讽谩骂，点踩暴涨
  - 平台发出永久封禁通知，吴伟账号被封，彻底崩溃落泪
  - 吴伟收到陌生号码电话/消息，被告知涉嫌寻衅滋事罪，网信办批准提起公诉
  - 《如愿》播放量飙升至千万级，退伍老兵话题全民热议，网友自发组织探访慰问当地老兵
- 章首钩子：吴伟目瞪口呆盯着《如愿》点赞数几何级暴增，脑袋嗡地一声——用对手视角的崩溃感开场，立刻抓住读者的爽感期待。
- 爽点：吴伟账号被永久封禁后又收到寻衅滋事公诉通知，双重暴击彻底踩死反派，读者爽感拉满。
- 章尾钩子：吴伟收到公诉通知、天旋地转瘫坐在地，与此同时《如愿》引发全民善意暖流，对比收尾，暗示江晨的影响力还在继续扩大。
- 出场角色：吴伟、江晨
- 字数目标：约 1885 字（原文实际）
