# 细纲（第 7 章）

### 第7章：和钟记者多联系就是工作
- 阶段位置：发展期 · 第1阶段第7章；完成专访并建立后续联系。
- 本章结构公式：江晨听见领导吹捧 + 正式接受采访 + 训练场参观 + 联系方式与身份悬念
- 本章禁止提前释放：采访文章的刊发效果、钟嘉嘉背景与后续传播推力；本章只建立联系。
- 核心事件：
  - 钟嘉嘉掏出笔记本准备深挖江晨事迹，江晨出现在门口，听到张耀祖和周薄森正在为自己辩护并大加夸赞
  - 周薄森将江晨介绍给钟嘉嘉，称其为「台柱子」，江晨对此哭笑不得
  - 钟嘉嘉正式表明身份——《军报》记者，与江晨握手，开始采访
  - 江晨接受采访，讲述视频构思、拍摄、剪辑、配音全过程，并将功劳推给领导，令两位团长老脸发红（实际上他们根本没看过视频）
  - 钟嘉嘉得知《亮剑》歌曲为江晨原创，感到更加惊讶，原以为是老前辈之作
  - 采访结束，张耀祖命江晨带钟嘉嘉参观训练场，士兵们纷纷以「江导」「晨哥」「江爷」称呼他
  - 傍晚钟嘉嘉婉拒饭局，告别离开；张耀祖替江晨要来她的联系方式并以军令形式交给江晨，称「和钟记者多联系就是工作」
  - 张耀祖独自判断钟嘉嘉来头不小（看了配车和随行兵），暗示她非普通实习生
- 章首钩子：江晨站在门口，亲耳听到张耀祖和周薄森正大力吹捧一个从未见过面的自己，制造喜剧悬念。
- 爽点：张耀祖亲自替江晨要来记者联系方式，以军令形式下达「和钟记者多联系就是工作」，上司逼着主角撩女神。
- 章尾钩子：张耀祖把钟嘉嘉电话塞给江晨并下达「军令」，留下「这个记者来头不简单」的悬念。
- 出场角色：江晨、钟嘉嘉、张耀祖、周薄森
- 字数目标：约 1579 字（原文实际）
