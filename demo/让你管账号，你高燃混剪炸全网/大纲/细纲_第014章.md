# 细纲（第 14 章）

### 第14章：“江晨，这次你完了！”
- 阶段位置：高潮期 · 第2阶段第4章；持续加压舆论危机并保留主角信息差。
- 本章结构公式：吴伟煽动质疑 + 粉丝争论升级 + 连续造谣获利 + 江晨仍专注创作
- 本章禁止提前释放：军方处置、江晨成片发布与舆论反转；本章只把危机推至热搜。
- 核心事件：
  - 吴伟在各大社交平台发布针对江晨的视频，煽动网友质疑其消费老兵、蹭热度
  - 质疑声涌入江晨抖手账号评论区，江晨粉丝与质疑者爆发激烈论战
  - 吴伟盯着涨粉数据得意，粉丝突破20万，决定趁热打铁继续制作视频
  - 江晨与钟嘉嘉在文工团电脑室讨论新歌MV制作，毫不知情外界风波
  - 吴伟发布第二条视频，暗示江晨有'背景'、靠后台获得官方资源
  - 吴伟再发视频造谣称'江晨是富二代，军校梦是父母花钱砸出来的'
  - '江晨是富二代'话题冲上热搜，营销号跟进铺天盖地深扒，粉丝群体陷入混乱
  - 吴伟粉丝突破50万，广告商主动联系洽谈合作，吴伟得意宣告'江晨这次你完了'
- 章首钩子：吴伟视频已是"重磅炸弹"级传播，但主角江晨对此一无所知——危机已至，当事人还在埋头创作。
- 爽点：吴伟粉丝数破50万、广告商主动送钱上门，反派看似大获全胜，衬托出主角即将面对的绝境翻盘张力。
- 章尾钩子：吴伟粉丝破50万、广告到手、高呼"江晨这次你完了"——主角尚未察觉，读者已知风暴已成。
