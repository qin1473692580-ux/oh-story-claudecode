# 题材风格包：悬疑脑洞型 / 死亡游戏 · 规则怪谈

> 番茄 / 七猫 / 黑岩通俗短篇。先读 `short-craft.md` 通用底座，本包在其上覆盖该题材专属调性。
> 一句话定位：**主角被卷入一套带死亡惩罚的诡异规则场景（怪谈村落 / 密室 / 死亡游戏），靠读懂规则字面意思、抓漏洞、识破人性陷阱通关求生。** 读者要的是「细思极恐」的智力恐惧 + 「读懂规则就能活」的解谜爽感，不是言情向的情绪宣泄。
> 数据依据：番茄公开题材趋势接口（fanqiehub.com/api/trending/dashboard，2026-07-16 抓取）显示"悬疑脑洞"是收录 120 本书的大类目，当期上升 29 本、下跌 16 本，净上升，验证这是当前平台上确实活跃且在涨的方向（这条是长篇分类口径下的数据，短故事口径的独立榜单未查到公开接口，仅作方向性佐证，不代表短故事子品类的精确数字）。

## 与通用底座的关键偏差（先看这条，别直接套 short-craft 情绪规则）

`short-craft.md` 默认「情绪宁烈不温、成段宣泄独白」，那是追妻火葬场 / 复仇打脸这类言情向短篇的调性。**悬疑脑洞型情绪基调不同**：不是心如死灰的委屈宣泄，是「细思极恐」的压抑智力恐惧——克制、冷静、字斟句酌地读规则，情绪靠"意识到规则背后的恶意"这个认知过程带动，不是直给的哭喊。

- 保留：第一人称在场叙述、章尾必留钩、超短节奏。
- 调整：情绪词密度降低，换成「后颈一凉」「后背发麻」「倒吸一口冷气」这类偏冷峻的克制反应；连环反问换成「逐条读规则、逐条推理」的解谜式内心独白。
- 保留（少见的言情式手法在这里依然适用）：向前剧透式钩子——但剧透的不是"他后来后悔了"，是"这条规则的真正含义直到第三天才会要命"这种悬念预告。

## 结构特征：关卡制合集，短故事的一种独立形态

真实案例《短故事系列之规则怪谈》（番茄小说，作者叶知秋灬，已完结，11.2万字，17章，分类"悬疑脑洞/悬疑/推理/规则怪谈"，来源：`fanqienovel.com/page/7486307675844643864`，2026-07-15核实页面存在）——**不是单篇一次性完结的短故事，是一部"关卡制合集"**：每一章换一个独立场景（槐荫村、幽闭疯人院、明德高中、凯撒大酒店、永昼购物中心……），场景之间人物不连续，只共享"规则怪谈"这个母题和写法，读者一章一个新地图、一套新规则，读完一章有独立的完结感，又会为下一章的新场景留住。

这是悬疑脑洞型区别于追妻火葬场那种"单篇一次性剧透到底"结构的地方：**合集式短故事**（多章各自独立小故事，共享同一写法母题）是这个题材常见的规模化写法，适合把好几个短故事点子攒成一部作品一次性投稿，而不是每个点子都单独开一篇。

作品简介原句（真实引用）："规则怪谈短故事系列以日常场景为茧，编织细思极恐的生存法则；各种场景充斥着各种恐怖悬疑的故事。记住：**所有温馨提示都是死亡预告的倒计时**！"——这句"温馨提示即死亡预告"本身就是这个题材最核心的钩子母题，见下节。

## 钩子母题与密度

- **温馨提示 = 死亡预告**：场景里任何"善意提醒"（前台交代的注意事项、墙上贴的告示、路人随口一句叮嘱）都是伪装过的致命规则，读者要学会对"看似正常的细节"保持警惕，这是本题材最大的爽点引擎——读者会自己提前猜"这句话是不是有诈"。
- **规则字面陷阱**：规则写的是字面意思，主角靠"抠字眼"活下来或者送命（"不能回头"没说"不能用镜子看"、"十二点前必须离开房间"没说"离开房间去哪"）。
- **规则反噬**：遵守了明面上的规则A，却因此触发了没写明的规则B，一次好心/守规矩反而致命，制造"越小心越危险"的压迫感。
- **人性抉择反转**：多人副本里，队友的背叛或自我牺牲是中段最大反转，读者要的是"猜不到谁会先叛变"。
- **场景本身就是钩子**：每一章换一个具体、有画面感的地点名（幽闭疯人院、末班公交、深夜地铁、第13号实验室），地点名自带悬念，读者靠地点名就能脑补一半恐惧。

## 情绪烈度与模式（克制冷峻型，非直给宣泄型）

- 用感官细节制造压抑感，不用直白情绪词轰炸：后颈一凉、后背泛起鸡皮疙瘩、呼吸放轻、手心全是冷汗、倒吸一口冷气。
- 推理式内心独白：不是连环反问式的情绪宣泄，是"如果……那……"的逐条排查式思考，读者跟着主角一起解谜。
- 短句制造窒息感：句子越短、停顿越多，压迫感越强，避免长句稀释紧张感。
- 反差萌调剂（可选招式）：主角面对绝境时插入一两句反差幽默或淡定台词（如"个人面板全是问号，开局吐血却淡定喊妈"），能缓冲长期高压带来的阅读疲劳，不是必须项，按人设决定用不用。

## 招式库

| 招式 | 怎么用 |
|------|--------|
| 温馨提示即死亡预告 | 任何场景内"善意提醒"都暗藏杀机，揭晓时制造"原来早有预兆"的后怕感 |
| 规则字面陷阱 | 主角或配角因为按常识理解规则（而非按字面）送命，反衬"读规则要逐字读"这条生存法则 |
| 规则反噬 | 遵守规则A触发未写明的规则B，制造"没有绝对安全区"的压迫感 |
| 人性抉择反转 | 多人副本里安排一次意料之外的背叛或牺牲，作为中段最大反转 |
| 场景切换制合集结构 | 每章独立地点+独立小规则，共享世界观母题，适合规模化产出，读者按章追更 |
| 反差萌淡定人设 | 主角对绝境反应异于常人（淡定/幽默），制造记忆点和传播梗 |

## 节奏骨架（单关卡内部）

```
进入新场景/新副本（章首意象预示或反差场景）
→ 拿到/发现规则清单（贴出具体规则条目，制造悬念）
→ 有人（配角/路人）无意间违反某条规则，当场示范死亡惩罚
→ 主角靠抠字眼/逆向推理找到规则漏洞或隐藏规则
→ 人性抉择时刻（信任危机/牺牲抉择）
→ 通关或逃出，代价与后怕感留白，钩向下一关卡/下一章
```

多关卡合集结构：上述骨架按章重复，人物设定和世界观母题贯穿全书，具体规则/地点每章更换。

## 收尾范式

- 通关但留创伤：活下来了，但失去了同伴或留下心理阴影，不写"云淡风轻翻篇"。
- 悬念钩向下一关：本章规则揭晓后，暗示"这不是最后一关"，为下一章/下一个副本埋伏笔。
- 反噬式黑色结局（少数篇章可用）：主角看似聪明地钻了规则空子，实则触发了更隐蔽的终极规则，制造意外感（慎用，避免让读者觉得"努力白费"太打击追读欲）。

## 该题材专属调校（与通用去 AI 规则 / 与追妻火葬场型的偏差）

- **情绪词密度要降**：short-craft 通用底座鼓励"心如死灰"式直给情绪词高频出现，本题材反而要克制，直给的委屈宣泄式情绪词放进规则怪谈会显得违和，换成感官化的冷峻反应（发凉、发麻、屏息）。
- **叙述姿态更接近推理小说，不是审判姿态**：追妻火葬场型允许叙述者带主观偏色审判对方（"我纯粹觉得你比较可笑"），本题材叙述者更多是"发现者"姿态，向读者展示线索和推理过程，不做情绪化的道德审判。
- **仍要避免的真·AI 味**：空洞的"气氛骤然变得诡异起来"这类无信息量的氛围总结句；论文体连词；规则条目本身可以形式化列出（这是本题材的功能性写法，不算 AI 味），但规则揭晓后不要再用一整段话复述解释一遍规则，解释应该融进推理动作里。
