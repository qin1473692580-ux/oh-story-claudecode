# 题材风格包：现实共鸣型 / 原生家庭剥削 · 职场PUA反杀 · 彩礼陷阱

> 番茄 / 七猫 / 黑岩通俗短篇。先读 `short-craft.md` 通用底座，本包在其上覆盖该题材专属调性。
> 一句话定位：**主角被至亲（父母/原生家庭）或权力方（职场上级/彩礼谈判对象）系统性剥削，用具体可量化的现实证据一步步坐实剥削事实，最终用现实手段（报警/起诉/曝光/经济independence）反制清醒离场。** 读者要的是"扎心的道德愤怒"+"清醒反击的解气感"，不是言情向的"意难平"。

> **本包证据强度说明（务必先读）**：本包依据两类来源写成——① 一篇公开创作分析文章（知乎，`zhuanlan.zhihu.com/p/1927364960729690189`，2026-07-15 用 Jina Reader 完整抓取核实全文）对这类题材的技法总结和一段引用示例；② 另一独立来源对"2025年高热关键词"的总结（监控证据/诊断书道具/婆媳算计等）与①方向一致，构成交叉印证。但**①文中引用的《无人兜底》"洪水弃女"这个具体案例，本包作者多次检索未能在番茄小说站内独立核实其确为已发表真实作品**，不排除是原作者自撰的示意性例句。因此下文涉及该案例的引用，明确标注"未独立核实，仅供技法参考"，不作为"这是真实爆款原文"的证据使用；本包其余技法总结部分有跨来源交叉印证，可信度相对更高。

## 与「追妻火葬场」型的关键区别（容易混淆，先辨清）

两者都是第一人称受害者向、情绪直给、爽点是"扎心+反击"，但内核不同，别套错公式：

| 维度 | 追妻火葬场型 | 现实共鸣型 |
|------|--------------|------------|
| 加害者 | 爱人/伴侣（背叛） | 至亲（父母/原生家庭）或权力方（上级/彩礼谈判对象） |
| 情绪基调 | 意难平的委屈+虐恋心死 | 扎心的道德愤怒+清醒的抽离 |
| 证据类型 | 撞破真相（偷听/偷拍私情） | 可量化的现实证据（诊断书/监控/聊天记录/彩礼账单/银行流水） |
| 反击手段 | 净身出户/转身嫁人/情感切割 | 现实工具：报警、公证、起诉、辞职信、家族群/社交平台曝光、经济独立离场 |
| 收尾情绪 | 心死后的平静或改嫁新生 | "我不再是提款机/工具人"式的清醒宣告，不追求圆满关系 |

## 叙述腔调

第一人称"我"全程主导，与通用底座一致；情绪直给，但**情绪母题是愤怒和清醒，不是心死委屈**。前半段可以有隐忍和困惑（"我一直以为这是为我好"），但转折点是"看清剥削的本质"而非"发现有第三者"，审判姿态更冷静克制、更讲道理，不是纯粹的反讽嘲弄。

---

## 开篇范式

前3秒必须砸出一个可量化、能截图的"命级/钱级"赌注——具体金额、具体伤害（如流产/工伤/失学）、具体证据物件，不铺垫背景。

范式（技法总结，非逐字真实引用）：

- **灾难/危机求救被拒**：主角在极端处境（自然灾害、意外、疾病）向本该保护自己的至亲求救，换来一句冷漠拒绝或转移话题的话，一句话坐实"这段关系不对等"。
- **重男轻女式量化对比**：具体列出资源分配的不公平数字（弟弟的学费/继承的房产份额/家务分工时长），用数字而非形容词制造实锤感。
- **职场PUA的话术开局**：上级用一句"为你好"式话术压低主角自我评价，紧跟一个具体的不合理要求（无偿加班/背锅/让渡成果）。
- **彩礼谈判的物化开局**：至亲把主角的婚姻当谈判筹码，开口谈的是具体金额和条件，不谈感情。

（本包作者提示：以上四条为技法结构总结，不是某一具体已发表作品的逐字引用；写作时应据此结构自行创作具体场景，不要把结构性描述当成可直接照抄的真实原文。）

## 钩子母题与密度

- **量化不公平清单**：用具体数字/物件反复对比"我"和被偏爱一方的待遇差异，数字比形容词更扎心。
- **求救被拒**：至亲/上级在主角最脆弱的时刻选择漠视或倒打一耙，是心理转折的导火索。
- **实锤证据链**：诊断书、监控录像、聊天记录截图、银行流水、录音——这类"物证"道具本身就是钩子，读者期待"证据什么时候被亮出来"。
- **公开处刑式反制**：把证据放到家族群、职场群、社交平台等公开场合曝光，用众人反应放大羞辱转移到加害者身上。
- **经济/法律工具反制**：报警、起诉、公证、辞职并保留证据——现实手段的具体流程感本身能增加可信度和爽感，不要写得含糊（"我报了警"不如"我把银行流水和聊天记录打印出来，一起交给了民警"）。

---

## 情绪烈度与模式

- 直白情绪词仍然保留（愤怒、心寒、恶心、麻木），但要接一个具体的证据物件或数字，不能空喊。
- 关键情绪是"清醒的抽离感"：从"我该怎么办才能让他们满意"切换到"我不欠任何人"，这个切换点要有一个具体触发事件（拿到关键证据、听到关键一句话），不能无来由地突然变坚强。
- 避免把加害者写成脸谱化的纯恶人：至亲式剥削往往裹着"为你好""都是一家人"的道德话术，这种话术的虚伪感才是让读者愤怒的核心，比单纯写恶毒更有共鸣。

## 招式库

| 招式 | 怎么用 |
|------|--------|
| 量化不公平对比 | 用具体数字（学费/继承份额/加班时长）反复对比资源分配差异，数字比形容词更扎心 |
| 求救被拒/倒打一耙 | 主角脆弱时刻求助，换来冷漠或反被指责，作为心理转折导火索 |
| 实锤证据链集齐 | 诊断书/监控/聊天记录/流水分批出现，最后一次性亮出形成反杀高潮 |
| 公开处刑式曝光 | 把证据放进家族群/职场群/社交平台，用旁观者反应放大羞辱转移 |
| 道德话术拆穿 | 复述加害者"为你好"式话术，再用一个具体事实戳破其虚伪，制造清醒感 |
| 现实工具反制 | 报警/起诉/公证/辞职离场，流程写具体，避免空泛带过 |

## 节奏骨架

```
命级/钱级赌注开局（具体金额/伤害/证据物件）
→ 量化不公平铺垫（数字对比，坐实长期剥削）
→ 求救被拒/倒打一耙（心理转折导火索）
→ 收集实锤证据链（分批出现，制造追读期待）
→ 清醒宣告（"我不欠任何人"式的定格句）
→ 现实工具反制（具体流程，不空泛带过）
→ 公开处刑或经济独立离场收尾
```

---

## 收尾范式

- 现实反制到位：报警/起诉/曝光后加害者受到具体后果（被拘留/被开除/被家族孤立），不写抽象的"恶有恶报"。
- 经济独立离场：辞职/搬离/断绝经济往来，强调"我靠自己"而非依赖新的关系。
- 清醒宣告式金句收尾：不是审判对方多可恶，是宣告"我不再是谁的提款机/工具人"，收束在主角自身的觉醒而非对方的悔恨。
- 慎用"对方追悔莫及"式收尾：这是追妻火葬场型的收尾模式，套进本题材容易让"至亲剥削"显得像可以被感动挽回的误会，削弱反制的彻底性。

## 该题材专属调校（与通用去 AI 规则的偏差）

- **保留**：直写情绪词+具体物件的通用底座写法；第一人称成段独白；审判金句式收尾。
- **调整**：审判对象从"变心的爱人"换成"剥削的至亲/权力方"，语气应更偏"清醒陈述事实"而非"讥讽嘲弄"，避免把父母/上级写成偶像剧脸谱式反派，虚伪的道德话术+具体的不公平事实并置，比单纯的恶毒台词更有真实感和共鸣度。
- **证据道具描写要具体**：诊断书写清楚病名和日期、聊天记录写清楚具体措辞、银行流水写清楚具体数字，模糊的"我拿到了证据"没有实锤感。
